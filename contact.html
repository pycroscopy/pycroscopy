Contact us¶

If you have any questions, find any bugs or if you want a feature added to pycroscopy, please see if an issue has already been raised with the same question and add your concerns to the most relevant issue. If you do not find an issue that is relevant, please raise a new issue. You will need a (free) Github account to do this. Wherever relevant, please provide the following information in your issue to help us resolve your issue quickly:

  • a (simplified) script / snippet that reproduces the error(s) you are facing

  • the full description of the error(s) you are seeing

  • python, pycrocopy, sidpy, and other relevant package versions etc.

  • details regarding your operating system (which version of Mac OS, Windows, Linux, etc.)
